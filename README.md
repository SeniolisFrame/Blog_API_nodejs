# Blog_API_nodejs

## Install :
### npm install

## ADD File :
#### .env file
###### ACCESS_TOKEN_SECRET = 'Your Access Token'
###### DB_URI = 'MongoDB URI'

## Start :
### npm start

## URL :
### /user
###### /user/register                     -> register id
###### /user/login                        -> login
###### /user/getProfile                   -> get my profile (require jwt)
###### /user/viewProfile/:id              -> get other profile
### /blog
